# 津軽為信の野望 改訂版

ブラウザで遊べる、津軽為信を題材にしたレトロ風シミュレーションゲームです。

## 遊び方

`index.html` をブラウザで開くと起動します。

GitHub Pagesで公開する場合は、このリポジトリの `main` ブランチのルートを Pages に指定してください。

## 内容

- 津軽為信を中心にした奥州統一シミュレーション
- 国取り、内政、外交、戦闘、季節進行
- クリア時のエンディングあり
- 単体HTMLで動作

## 注意

このゲームは試作品・改訂版です。
ゲームバランスや細部の演出は今後調整する可能性があります。
